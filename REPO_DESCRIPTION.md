# Descrição do repositório (GitHub)

Use o texto abaixo no campo **Description** do repositório. Máximo **350 caracteres**.

---

## Opção recomendada (≈ 290 caracteres)

**Português:**

Protocolo aberto para o domínio da saúde. Observa sinais, prioriza com equidade e orquestra cuidado (insumos, leitos, atenção) com soberania do dado e transparência determinística. O UHP é para a saúde conectada o que o HTTP foi para a web. Domínio `u.health` sobre Open USP (dados); entrega física via Open UDP.

**English:**

Open protocol for the health domain. Observes signals, prioritizes with equity, and orchestrates care (supplies, beds, attention) with data sovereignty and deterministic transparency. UHP is to connected health what HTTP is to the web. `u.health` domain on Open USP (data); physical delivery via Open UDP.

## Opção longa (até 350 caracteres)

**Português:**

Open UHP (Universal Health Protocol): protocolo aberto para saúde. Define como sinais são observados, priorizados (Índice de Prioridade) e respondidos com equidade algorítmica, soberania do dado (LGPD, privacidade diferencial) e trilha imutável. Cinco Leis Imutáveis. Domínio `u.health` sobre Open USP; entrega física Open UDP. Especificação completa, MIT.

**English:**

Open UHP (Universal Health Protocol): open protocol for health. Defines how signals are observed, prioritized (Priority Index), and answered with algorithmic equity, data sovereignty (differential privacy), and an immutable trail. Five Immutable Laws. `u.health` domain on Open USP; Open UDP physical delivery. Full specification, MIT.
