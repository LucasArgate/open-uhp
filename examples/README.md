# Open UHP — Exemplos

Esta pasta destina-se a exemplos de implementação e uso do protocolo Open UHP (`u.health`).

## Estrutura planejada

- **basic/** — Fluxo mínimo: GRANT_CONSENT → OBSERVE_SIGNAL → ASSESS_PRIORITY → ALLOCATE_RESOURCE → UPDATE_STATUS (COMPLETED).
- **advanced/** — RAISE_ALERT (surto/ruptura/fraude), bypass de emergência (L1), privacidade diferencial, operação offline-first (L5), integração FHIR.
